Sobre o SIRA

Nascemos de uma iniciativa Acadêmica!!

Projeto Integrador Univesp (Universidade Virtual do Estado de São Paulo)

PROBLEMA DO PROJETO: As secretarias das escolas públicas não possuem um sistema específico e apropriado para o registro dos atendimentos realizados e o acompanhamento dos diversos atendimentos escolares relacionados aos alunos.

OBJETIVO DO PROJETO: Desenvolver um sistema para cadastro de atendimentos que contemple informações como: nome do requerente, tipo de serviço solicitado, nome do atendente, situação da demanda solicitada, dentre outras. Além disso, a solução deve contemplar a consulta através de uma banco de dados dos atendimentos realizados.

SOLUÇÃO ENCONTRADA: Desenvolver uma aplicação web para gerenciamento dos atendimentos realizados aos pais e alunos da rede pública de ensino de Rio Claro.
